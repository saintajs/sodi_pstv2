Web Progress 
==================
Módulo para Odoo 17 que mejora la experiencia de importación de datos mostrando una barra de progreso visual con múltiples estilos (Estándar, Nyancat y Moderno) durante la carga y procesamiento de archivos.

Características
--------------
* Muestra una barra de progreso visual durante la subida de archivos y la importacion de datos
* Muestra una barra de progreso en toda accion que tenga cierto tiempo de carga 
* Tres estilos diferentes de barras de progreso:
  * **Estándar**: Muestra solo el porcentaje de progreso
  * **Nyancat**: Barra de progreso animada con el gato de Nyan
  * **Moderno**: Diseño limpio y moderno con barra de progreso
* Fácil de usar: Cambia entre estilos con un menú desplegable
* Compatible con todas las operaciones de importación de Odoo

Configuración
============
* No se requiere configuración adicional.
* El módulo se activa automáticamente después de la instalación.

Instalación
-----------
1. Copie el directorio `web_progress_sunni` a la carpeta de complementos de Odoo
2. Actualice la lista de aplicaciones en Odoo
3. Busque e instale "Web Progress Sunni"
4. ¡Listo! La próxima vez que importe datos, verá la nueva barra de progreso

Uso
---
1. Vaya a cualquier vista de lista en Odoo
2. Haga clic en "Importar"
3. Seleccione un archivo para importar
4. Use el menú desplegable en la esquina superior derecha para cambiar entre los diferentes estilos de barra de progreso
